# Paint
### How to compile and run HelloPaint and Paint
#### Windows
For the sake of having a reasonable file size, SFML is not included with the file, you will have to download it yourself:

- Download SFML 3.0.2.
- Inside the .zip file you download, there should be a folder called SFML-3.0.2.
##### HelloPaint:
- Move the SFML folder to a directory with the HelloPaint .cpp file
##### Paint: 
- Move the SFML folder to the Paint folder containing the main.cpp file and other folders.
##### Both:
- To build in your terminal (Command Prompt, PowerShell):
```
mingw32-make -f Makefile.txt
```
